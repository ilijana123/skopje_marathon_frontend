# Skopje Marathon Frontend

This project is built with **React + Vite + TypeScript** and communicates with [Spring Boot REST API - Java](https://github.com/ilijana123/skopje_marathon).

## Getting Started

### Install dependencies
```bash
npm install
'''
Run the development server
bash
npm run dev
Open:
http://localhost:5173

Backend API
The frontend expects a backend running at:
http://localhost:8081
API calls are proxied via Vite using /api.
